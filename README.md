# Ghost Story Work (괴담출근) Universe

This repository serves as the central workspace for the "Ghost Story Work" universe projects. It contains the worker agent and the related project repositories.

## Structure

- **[universe_agent](./universe_agent)**: The AI Research & Content Generation Agent.
- **[dreamcorp-intranet](./dreamcorp-intranet)**: The Dream Corporation Intranet project.
- **[bureau-of-supernatural-disaster](./bureau-of-supernatural-disaster)**: The Bureau of Supernatural Disaster project.

## Usage

To use the agent to manage these projects:

```bash
# Run the agent to generate content
python universe_agent/main.py compose "Character Name" --type "Letter" ...
```
