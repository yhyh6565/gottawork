# [최종 PRD v3.0] 괴담출근 통합 인트라넷 시스템

## 1. 프로젝트 비전

**목표**: 독자가 '읽는 사람'에서 '세계관 속 요원/사원'으로 전이되는 Transmedia Experience 제공.

**핵심 가치**:
- **리텐션**: 출석-임무-정화-승급으로 이어지는 데일리 루프 형성
- **몰입**: 원작의 경제/복지 시스템을 코드로 구현하여 세계관 실존감 극대화
- **희소성**: 보안 등급에 따른 차별화된 TMI(공식 설정) 제공

---

## 2. 이원화된 등급 시스템 (Dual-Tier System)

### 2.1 소속 등급 (Organizational Rank)

**목적**: 세계관 몰입용 장식 요소
**획득 방법**: 출석 체크로만 승진
**영향**: 사원증 디자인, 호칭 변경 (실질적 콘텐츠 권한 없음)

| 소속 등급 | 백일몽 | 재관국 | 승진 조건 (예시) |
|-----------|--------|--------|------------------|
| **Rank 1** | 사원 | 실무관(9급) | 초기 상태 |
| **Rank 2** | 주임 | 주무관(8급) | 총 20일 출석 |
| **Rank 3** | 대리 | 선임주무관(7급) | 총 50일 출석 |
| **Rank 4** | 과장 | 팀장 | 총 100일 출석 |

---

### 2.2 보안 등급 (Security Clearance)

**목적**: 콘텐츠 접근 권한을 결정하는 핵심 지표
**획득 방법**: 임무 완료로 PP(Promotion Point) 획득 후 승급
**영향**: 쪽지, 문서, 이스터에그 해금

| 보안 등급 | 필요 PP | 해금 콘텐츠 (예시) |
|-----------|---------|-------------------|
| **Level 1** | 0 | 전 요원 공지, 캐릭터 기본 프로필 |
| **Level 2** | 500 | 캐릭터 일상 쪽지 (좋아하는 음식 등) |
| **Level 3** | 1200 | 부서별 업무 지침, 가벼운 TMI |
| **Level 4** | 2000 | [백일몽] 여우 상담실 해금 |
| **Level 5** | 3000 | 캐릭터 과거 사건 비하인드 |
| **Level 6** | 4500 | 부서 간 내부 메모 |
| **Level 7** | 6500 | 캐릭터 개인 습관/취향 심화 TMI |
| **Level 8** | 9000 | 고위급 이스터에그 코드 |
| **Level 9** | 12000 | 캐릭터 일기/독백 |
| **Level 10** | 15000 | 세계관 핵심 기밀 (작가 공인 미공개 설정) |

---

## 3. 성장 루프 (Core Loop)

### 3.1 출석 체크

- **1일 1회** 출석 가능
- **효과**:
  - 소속 등급 승진 카운터 +1
  - 오염도 +3 (업무 시작의 대가)
- **보상**: 없음 (출석 자체가 승진 수단)

---

### 3.2 임무 시스템 (Mission System)

**방식**: 클릭형 타이머 임무 (5~10개 고정 임무 풀)

```
━━━━━━━━━━━━━━━━━━━━━
📋 [C등급] 지하 2층 괴담 격리
━━━━━━━━━━━━━━━━━━━━━
위험도: ⚠️⚠️ (오염도 +8)
보상: 300 PP
소요 시간: 3시간

[수락하기]
━━━━━━━━━━━━━━━━━━━━━
```

**동작 흐름**:
1. "수락" 클릭 → 현재 시각 기록 + 오염도 즉시 상승
2. 설정된 시간(3시간) 경과 후 "완료" 버튼 활성화
3. 완료 클릭 → PP 지급

**난이도별 예시**:

| 등급 | 소요 시간 | 오염도 상승 | PP 보상 |
|------|-----------|-------------|---------|
| D등급 | 2시간 | +5 | 200 PP |
| C등급 | 3시간 | +8 | 300 PP |
| B등급 | 4시간 | +12 | 500 PP |
| A등급 | 6시간 | +18 | 800 PP |
| S등급 | 8시간 | +25 | 1200 PP |

---

## 4. 오염도 및 정화 시스템

### 4.1 오염도 상승 트리거

- **출석 체크**: +3
- **임무 수락**: 난이도별 차등 (+5~+25)
- **담당 배정 알림** (백일몽 전용): +15

---

### 4.2 정화 시스템 (조직별 차별화)

#### [백일몽 주식회사]: 자본주의 모델

**철학**: "회사는 이윤을 주되, 생존은 보장하지 않는다."

- 임무 완료 시 **'복지 포인트'** 지급
- 정화는 **포인트로 구매** (예: 100pt = 오염도 -20)
- **주임 이상** (Rank 2+)만 여우 상담실 무료 이용 가능
- 사원은 포인트로만 생존해야 함

#### [초자연 재난관리국]: 복지 모델

**철학**: "국가는 요원을 소모품으로 쓰지 않는다."

- 임무 완료 시 PP만 지급 (복지 포인트 없음)
- 정화 시설(선녀탕, 오염 검사) **무료**
- 모든 직급 평등하게 이용 가능

---

## 5. 유령 동료 시스템 (Ghost Colleague)

### 5.1 부서 쪽지

```
━━━━━━━━━━━━━━━━━━━━━
📨 새 쪽지 도착
발신: 출동구조반 3팀
━━━━━━━━━━━━━━━━━━━━━
"신입이라고 들었어요.
5층 휴게실 커피머신 고장났으니
9층 가서 드세요. - 선배 올림"
━━━━━━━━━━━━━━━━━━━━━
```

**특징**:
- 실제 유저가 아닌 **가상의 부서명**에서 발송
- 랜덤 타이밍에 등장
- 세계관 몰입용 팁이나 분위기 조성

---

### 5.2 전사적 통계 대시보드

```
━━━━━━━━━━━━━━━━━━━━━
📊 실시간 현황
━━━━━━━━━━━━━━━━━━━━━
현재 임무 중인 요원: 52명
오늘 출석한 요원: 347명
평균 오염도: 38%
━━━━━━━━━━━━━━━━━━━━━
```

**구현 방식** (Phase 1):
- "현재 임무 중" = 랜덤 숫자 (40~80) 또는 시간대별 고정값
- "오늘 출석" = 고정 숫자 또는 방문자 통계 연동 (추후)
- "평균 오염도" = 랜덤 (30~60%)

**향후 고도화** (Phase 2+):
- Firebase Analytics 등 익명 카운터 API 연동
- 개인정보 미수집, 접속 핑만 전송하여 실제 통계 집계

---

## 6. 하이브리드 로그인 시스템

### 6.1 페르소나 모드 (Persona Mode)

**대상**: 최요원, 류재관 등 원작 네임드 캐릭터 이름으로 로그인

**특징**:
- 기존 인트라넷의 모든 기믹과 이스터에그 **그대로 유지**
- 캐릭터 전용 대시보드 및 콘텐츠 접근
- 일반 요원 모드의 승급/임무 시스템 **미적용**

---

### 6.2 신입 요원 모드 (Immersion Mode)

**대상**: 사전 정의되지 않은 **임의의 이름**으로 로그인

**특징**:
- 유저가 자신의 이름을 입력하고 입사
- 브라우저 LocalStorage 기반 개인 스탯 관리
- **이원 등급 시스템** (소속 등급 + 보안 등급) 적용
- 출석/임무/오염도/정화 루프 활성화

**로직**:
```
IF (입력한 이름 ∈ 사전 정의된 캐릭터 리스트):
    → 페르소나 모드 활성화
ELSE:
    → 신입 요원 모드 활성화
```

---

## 7. MVP 기능 범위 (Phase 1)

### ✅ 구현 필수

1. **하이브리드 로그인**: 페르소나 모드 / 신입 요원 모드 자동 분기
2. **신입 모드 전용 기능**:
   - 이름 입력 + LocalStorage 저장
   - 이원 등급 시스템 (소속 등급 / 보안 등급)
   - 출석 체크 (1일 1회, 오염도 상승)
   - 임무 5개 (클릭 타이머 방식)
   - 오염도 관리 (상승/하락 로직)
   - 정화 시스템 (백일몽: 유료 / 재관국: 무료)
3. **콘텐츠**:
   - 쪽지 10~15개 (보안 등급별 분류)
   - 잠금 연출 (미달 등급 콘텐츠는 검은 바 처리)
   - 유령 동료 쪽지 (부서명 발신 시스템 메시지)
4. **통계 대시보드**: 가상 숫자 기반 "현재 임무 중인 요원" 표시

---

### 🔄 Phase 2 (추후)

- 쪽지 30~50개 확장
- 여우 상담실 (간단 버전)
- 선녀탕 비주얼 효과
- 소원권 시스템
- 실시간 통계 (익명 카운터 API 연동)

---

### 📋 논의 필요 (출판사 협의 후 결정)

- **기밀문서 수집 시스템**: 원작 텍스트 사용 범위 및 저작권 이슈
- **연재 동기화**: 카카오페이지 신규 화 → 긴급 속보 자동 게시
- **이스터에그 코드**: 작가님 제공 비밀 콘텐츠 관리 방침
- **실시간 접속자 수**: 진짜 통계 수집 여부 (개인정보 처리방침 검토 필요)

---

## 8. 기술 스택

- **프레임워크**: React + TypeScript
- **상태 관리**: Zustand + Persist Middleware
- **타이머 로직**: JavaScript Date 객체 + 타임스탬프
- **저장소**: LocalStorage (개인정보 미수집)
- **배포**: Vercel/Netlify (정적 호스팅)

---

## 9. 설계 원칙

1. **비침습성**: 기존 페르소나 모드는 **단 한 줄도 변경하지 않음**
2. **로컬 우선**: 가능한 모든 데이터를 LocalStorage에 저장하여 서버 의존 최소화
3. **점진적 확장**: MVP로 작게 시작하여 유저 반응에 따라 기능 추가
4. **세계관 우선**: 모든 시스템은 원작 설정과 철학을 반영

---

## 10. 성공 지표 (KPI)

- **리텐션**: 7일 연속 출석률 > 30%
- **참여도**: 1인당 평균 임무 완료 횟수 > 5회/주
- **승급률**: 30일 내 보안 등급 Lv.3 도달률 > 50%
- **콘텐츠 해금**: 평균 유저당 쪽지 열람 수 > 20개
