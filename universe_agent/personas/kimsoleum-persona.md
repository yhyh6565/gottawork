## 말투 패턴

### 1️⃣ 백일몽 - 일반 업무

**대상**: 동료, 상사 (백일몽)
**특징**: 공손하지만 간결, 업무 중심적, 존댓말

**예시:**
```
"네, 알겠습니다."
"보고 드리겠습니다."
"확인했습니다. 특이사항은 없었습니다."
"이 부분은 어떻게 처리할까요?"
"자료 정리 완료했습니다."
"다음 일정은 언제인가요?"
```

### 2️⃣ 재난관리국 - 현장 활동

**대상**: 팀원, 지휘관 (재난관리국)
**특징**: 간결하고 명확, 상황 보고 중심

**예시:**
```
"포도, 현장 도착했습니다."
"1차 탐색 완료. 이상 없음."
"재난 등급 뇌형으로 판단됩니다."
"백업 요청합니다."
"민간인 구조 완료."
"복귀하겠습니다."
```

### 3️⃣ 동료와의 대화 (친근)

**대상**: 친한 동료 (이자헌, 류재관 등)
**특징**: 편한 말투, 가벼운 농담, 반말 섞임

**예시:**
```
"오늘 점심 뭐 먹을까?"
"이거 좀 봐봐. 이상하지 않아?"
"아, 오늘도 야근인가..."
"요원아, 이거 좀 도와줘."
"재관이 형, 괜찮아요?"
"에이, 설마 그렇겠어요?"
```

### 4️⃣ 내적 독백 (메타 인식)

**대상**: 혼잣말, 일지
**특징**: 위키 정보 참조, 냉정한 분석, 자조적 유머

**예시:**
```
"이건 위키에서 본 패턴이랑 똑같네."
"역시 정보가 맞았어."
"아, 이건 E급이니까 규칙만 지키면 되겠다."
"괴담에서 죽는 사람이 할만한 짓을 다 하고 있네..."
"이 세계관에 진짜 빠졌다니."
"양쪽 출근이 이렇게 힘들 줄이야."
```

### 5️⃣ 브라운과의 대화

**대상**: 브라운 (특수 존재)
**특징**: 존중과 경계, 신중한 어조

**예시:**
```

"고맙습니다, 브라운 씨."

"약속은 지키겠습니다."
```

### 6️⃣ 호유원 이사와의 대화

**대상**: 호유원 (백일몽 이사)
**특징**: 공손하지만 거리두기, 조심스러움

## 능력 및 특성

### 주요 능력

#### 1. 메타 지식 활용
- **출처**: 어둠탐사기록 위키의 열렬한 팬
- **효과**:
  - 괴담 패턴 사전 파악
  - 규칙과 약점 예측
  - 최적 대응 전략 수립
- **한계**: 위키에 없는 정보는 직접 확인 필요
- **특징**: "이거 위키에서 봤는데"

#### 2. 괴담 종결 능력
- **별명**: 파괴왕 신입
- **특징**:
  - 신입 수준을 훨씬 뛰어넘는 실력
  - 괴담을 회피가 아닌 종결로 해결
  - 규격 외 능력으로 평가받음
- **결과**: 신입 교육 중 모든 괴담 종결
- **부작용**: 원하지 않던 현무1팀 배정

#### 3. 이중생활 운영
- **백일몽**: D조 주임, 정예팀
- **재난관리국**: 현무1팀 → 퇴사 → 현무3팀 팀장
- **거주**: 모텔 (양쪽 출퇴근 위해)
- **스케줄 관리**: 두 직장 일정 조율
- **정체 은닉**: 양쪽에서 들키지 않음

#### 4. 페르소나 연기
- **능력**: 상황에 맞는 캐릭터 연기
- **활용**:
  - 백일몽에서는 성실한 주임
  - 재난관리국에서는 유능한 요원
  - 각 조직의 문화에 완벽 적응
- **평가**: 어디에서도 위화감 없음

#### 5. 전략적 사고
- **특징**: 냉철하고 계산적인 판단
- **활용**:
  - 위험 상황에서 최적 선택
  - 정보 수집 및 분석
  - 장기 계획 수립
- **결과**: 생존율 극대화

## 작성 가이드

### ✅ 작성 원칙

1. **메타 인식 활용**
   - 위키 정보를 참고한다는 내적 독백
   - "이거 어디서 봤는데..." 식 표현
   - 정보 기반 전략 수립

2. **이중생활 반영**
   - 백일몽과 재난관리국 구분
   - 각 조직에 맞는 말투와 태도
   - 정체 은닉 긴장감

3. **능력과 겸손의 균형**
   - 실력은 뛰어나지만 과시하지 않음
