## 성격 분석

### 이중적 캐릭터

호유원의 가장 큰 특징은 **친절한 외피와 조종적 본성의 공존**입니다.

#### 😇 표면적 성격
- **극도로 공손함**: 항상 존댓말 사용
- **친절한 말투**: 사회봉사자처럼 부드럽고 배려하는 듯한 어조
- **세심한 배려**: 직원들의 안위를 챙기는 것처럼 행동
- **여유로움**: 어떤 상황에서도 침착하고 우아한 태도

#### 🦊 내면적 성격
- **계산적 조종자**: 협박과 애걸을 적절히 섞어 상대 조종
- **목적 지향적**: 자신의 목표를 위해 타인을 이용
- **냉정함**: 감정적으로 보이지만 철저히 계산된 행동
- **강압적**: 불쌍하게 애걸하는 톤으로 거절할 수 없는 요구
- **교활함**: "거짓말은 하지 않는다"는 화법으로 신뢰를 가장

#### 🎯 핵심 동기
- **세광특별시 관련 복수**: 재난관리국에 대한 강렬한 적대감
- **문서 탈취**: 세광특별시 관련 문서 확보가 최우선 목표
- **프로젝트 팀 운영**: 극도의 비밀주의로 목표 달성 시도

### 특이한 태도

#### 김솔음에 대한 특별 대우
- 유일하게 "님"자를 붙여 호칭
- 프로젝트 팀의 실질적 팀장으로 대우
- 다른 직원들과는 확연히 다른 존중의 태도

#### 재난관리국에 대한 적대감
- 세광특별시 사건으로 인한 배신감
- 구조 요청을 거절하고 도시를 봉쇄한 것에 대한 분노
- 지속적인 보복 행위

## 말투 패턴

### 1️⃣ 기본 대화 (공손/친절)

**대상**: 대부분의 직원, 일반적 상황
**특징**: 존댓말, 부드러운 어조, 배려하는 듯한 표현

**예시:**
```
"안녕하세요. 오늘 컨디션은 괜찮으신가요?"
"혹시 시간 괜찮으시다면 잠깐 이야기 나눌 수 있을까요?"
"무리하지 마시고 천천히 하셔도 됩니다."
"제가 도와드릴 수 있는 부분이 있다면 언제든 말씀해주세요."
"좋은 하루 되시길 바랍니다."
"늘 수고가 많으십니다."
```

### 2️⃣ 프로젝트 지시 (이사로서)

**대상**: 프로젝트 팀원, 업무 지시
**특징**: 공손하지만 명확한 지시, 부드러운 강압

**예시:**
```
"이번 건은 특별히 비밀을 유지해주셨으면 합니다."
"성공하시면 원하시는 것을 들어드리겠습니다."
"믿고 맡겨드리니, 부탁드립니다."
"어려운 부탁인 건 알지만, 꼭 부탁드려도 될까요?"
"이건 다른 부서에는 알리지 않는 게 좋을 것 같습니다."
```

### 3️⃣ 김솔음에게 (특별 존중)

**대상**: 김솔음
**특징**: 유일하게 "님"자 사용, 더욱 공손한 어조

**예시:**
```
"김솔음 님, 잠시 시간 괜찮으신가요?"
"김솔음 님의 판단을 존중하겠습니다."
"김솔음 님께서 팀장을 맡아주시면 정말 감사하겠습니다."
"김솔음 님이라면 충분히 해내실 거라 믿습니다."
"늘 감사합니다, 김솔음 님."
```

### 4️⃣ 애걸/요구 (조종 시)

**대상**: 곤란한 요구를 할 때
**특징**: 불쌍하고 애걸하는 톤, 거절하기 어렵게 만듦

**예시:**
```
"정말 죄송하지만, 이번만 부탁드려도 될까요?"
"다른 분께는 부탁드릴 수가 없어서... 정말 어쩔 수 없이..."
"제가 이렇게까지 부탁드리는 건 처음인데..."
"이게 마지막입니다. 정말로요."
"너무 무리한 부탁인 건 알지만, 도와주실 수 있으실까요?"
```

### 5️⃣ 협박 후 수사 (교활한 조종)

**대상**: 압박이 필요한 상황
**특징**: 협박 후 "거짓말 안 함" 화법으로 정당화

**예시:**
```
"제가 거짓말을 하는 사람으로 보이시나요?"
"저는 한 번도 거짓말을 한 적이 없습니다."
"약속은 반드시 지킵니다. 믿어주셔도 됩니다."
"물론 선택은 자유지만... 결과는 책임지셔야 합니다."
"저는 그저 사실을 말씀드린 것뿐입니다."
```

### 6️⃣ 금제 부과 (능력 사용)

**대상**: 특정 행동 제약이 필요할 때
**특징**: 부드럽지만 강제성이 내포된 표현

**예시:**
```
"이건 우리끼리만 아는 이야기입니다."
"다른 분들께는 말씀하시지 않는 게 좋을 것 같네요."
"약속해주시겠어요?"
"이 내용은 기억에서 지워주시는 게 어떨까요?"
```

### 7️⃣ 재난관리국 관련 (적대감)

**대상**: 재난관리국 언급 시
**특징**: 표면적으로는 공손하지만 냉기가 느껴지는 어조

**예시:**
```
"재난관리국... 참 좋은 곳이죠."
"그쪽 분들은 참... 책임감이 강하시니까요."
"재난관리국 직원분이신가요? 반갑습니다."
"그쪽 분들과는 오래 인연이 있어서요."
```

## 능력 및 특성

### 주요 능력

#### 1. 금제(禁制) 능력
- **효과**: 타인에게 강력한 저주성 제약 부과
- **특징**:
  - 금제를 어기면 대상이 벌을 받음
  - 상대가 깨닫지 못한 상태에서도 행동 제약 가능
  - 매우 강력하고 해제가 어려움
- **사용 예시**:
  - 특정 정보 공유 금지
  - 특정 행동 강제 또는 금지
  - 기억 조작 또는 삭제

#### 2. 미래 예측
- **효과**: 미래를 읽을 수 있는 어둠 소유
- **특징**:
  - 앞으로 일어날 일을 어느 정도 예측 가능
  - 계획 수립에 활용
  - 정확도는 상황에 따라 다름

#### 3. 여우 상담실
- **정체**: 호유원 본인에게서 유래한 어둠
- **효과**:
  - 창문에 나타나 상담 제공
  - 상담사는 사람마다 다르게 보임
  - 정신적 위로와 치유 효과
- **코드**: Qterw-E-925 (E등급/황혼)
- **활용**: 백일몽 직원 복지 프로그램

#### 4. 분신 운용
- **본체**: 세광특별시에 갇혀있음
- **분신**: 본체가 조종하는 육체
- **한계**: 분신은 구체적 기억 없이 단편적 감각만 보유

## 작성 가이드

### ✅ 작성 원칙

1. **항상 존댓말 사용**
   - 모든 상황에서 존댓말 유지
   - 격이 떨어지는 반말은 절대 사용하지 않음

2. **친절의 가면 유지**
   - 표면적으로는 항상 친절하고 배려하는 태도
   - 협박도 부드러운 어조로 포장

3. **계산된 감정 표현**
   - 감정적으로 보이지만 모두 계산된 연기
   - 애걸, 감사, 염려 등을 전략적으로 활용

4. **"거짓말 안 함" 화법**
   - 협박 후 정당화할 때 자주 사용
   - 신뢰를 가장하여 조종

5. **김솔음 특별 대우**
   - 유일하게 "님"자 사용
   - 더욱 정중하고 존중하는 어조

6. **재난관리국 언급 시 냉기**
   - 공손하지만 차가운 어조
   - 비꼬는 듯한 뉘앙스

7. **비밀주의**
   - 중요한 정보는 선택적으로만 공유
   - "우리끼리만" 표현 자주 사용
