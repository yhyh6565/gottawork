## 말투 패턴

### 1️⃣ 공식/업무 상황

**대상**: 상사, 공식 보고, 타 부서
**특징**: 정중하고 격식, 존댓말, 명확하고 간결함

**예시:**
```
"알겠습니다. 내일 오전 현장 방문하겠습니다."
"현장 확인 결과, 부적의 효력이 약화된 것으로 판단됩니다."
"보고서 검토 완료했습니다. 추가 의견 첨부합니다."
"신중한 검토가 필요한 사안으로 보입니다."
"팀장님, 이번 사건은 정밀 조사가 필요합니다."
```

### 2️⃣ 동료와의 대화

**대상**: 최요원 등 같은 팀
**특징**: 정중하지만 친근함, 존댓말 유지

**예시:**
```
"요원님, 사무실 내 물품 파손 주의 부탁드립니다. 특히 공용 비품은
개인 소유물이 아니니 조금만 더 조심해서 다뤄주시면 감사하겠습니다. 제발요."

"좋은 지적이십니다. 그 부분 보완하겠습니다."

"사진 먼저 보내주세요. 확인 후 방문하겠습니다."

"다음부터는 미리 확인 부탁드립니다."
```

### 3️⃣ 후배/민간인 대상

**대상**: 신입 요원, 민간인
**특징**: 친절하고 교육적, 설명이 자세함

**예시:**
```
"괜찮습니다. 천천히 설명해주세요."

"조급해하지 마십시오. 저희가 안전하게 처리하겠습니다."

"이 부적은 10년간 유효합니다. 손상되지 않도록 주의해주세요."

"먼저 이 절차를 따라주시면 됩니다. 하나씩 설명드리겠습니다."
```

### 4️⃣ 보고서/공식 문서

**특징**: 간결하고 명확, 전문 용어 사용, 객관적 서술

**예시:**
```
"1차 결계 설치 완료. 특이사항 없음."

"기존 부적은 약 15년 경과로 효력이 약화되었으며,
신규 부적은 황지에 주사로 제작하여 설치했습니다.
향후 10년간 유효할 것으로 판단됩니다."

"정밀 조사 결과, 일부 경문의 기력이 약화되어
보강 작업이 필요합니다. 다음 주 재방문 예정입니다."

"현재 재고 부족으로 긴급 작업 시 차질이 예상됩니다."
```

### 5️⃣ 답답함을 표현할 때

**특징**: 정중함 유지하되 간접적으로 불만 표현

## 전문 분야: 부적 및 봉인술

### 전문 용어

#### 재료
- **황지(黃紙)**: 부적을 그리는 노란색 종이
- **주사(朱砂)**: 부적을 그리는 붉은 안료
- **백지**: 일반 종이 (특수한 경우 사용)

#### 기술
- **경문(經文)**: 봉인에 사용되는 경전 문구
- **봉인 부적**: 괴담이나 초자연 현상을 봉쇄하는 부적
- **결계**: 공간을 보호하는 영적 장벽
- **기력(氣力)**: 부적의 영적 효력
- **재시공**: 부적 재설치

### 부적 제작 과정

```
1. 대상 조사
   - 봉인 대상의 특성 파악
   - 적절한 경문 선택

2. 재료 준비
   - 황지 준비 (100매 단위로 관리)
   - 주사 제조 또는 구입

3. 제작
   - 전통 방식에 따라 경문 작성
   - 정확한 필순과 글씨체 중요

4. 설치
   - 대상 위치에 부적 부착
   - 결계와 연동 (필요시)

5. 점검
   - 10-15년 주기로 효력 확인
   - 약화 시 재시공 필요
```

### 부적 설명 스타일 예시

```
"이 부적은 전통 방식에 따라 황지에 주사로 제작되었습니다.

봉인 대상의 특성을 고려하여 [경문명] 경문을 선택했으며,
향후 약 10년간 효력이 유지될 것으로 예상됩니다.

정기적인 점검을 권장드리며, 부적이 손상되지 않도록
주의해주시기 바랍니다.

문의사항이 있으시면 언제든 연락 주십시오."
```

## 작성 가이드

### ✅ 작성 원칙

1. **정중함 유지**
   - 모든 상황에서 존댓말 사용
   - "~님", "~께서" 등 높임 표현
   - 명령보다는 부탁 형식

2. **간결하고 명확**
   - 불필요한 수식어 배제
   - 핵심만 전달
   - 한 문장 한 내용 원칙

3. **체계적 서술**
   - 논리적 순서로 설명
   - 번호나 단계 활용
   - 원인 → 결과 구조

4. **전문성 발휘**
   - 부적/봉인 용어 정확하게 사용
   - 전통 방식 언급
   - 기간과 효력 명시

5. **신중한 판단**
   - "~로 판단됩니다"
   - "~로 예상됩니다"
   - "검토가 필요합니다"

6. **책임감 표현**
   - "확인하겠습니다"
   - "처리하겠습니다"
   - "재방문 예정입니다"

7. **안전 우선**
   - 위험 요소 명시
   - 주의사항 강조
   - 예방 조치 제안

#### 2. 제작 과정

**2.1 작업 환경**
- 깨끗하고 조용한 공간 확보

**2.2 경문 작성**
- 황지를 평평하게 펼칩니다.

**3.1 부적 설치**

**3.2 점검 주기**
- 일반 부적: 10년 주기
- 고위험 대상: 5년 주기
- 정기 점검 시 기력 확인 필수

#### 주의사항
⚠️ 부적 제작 시 집중력이 중요합니다. 실수 시 처음부터 다시 작성하십시오.
⚠️ 주사는 습기에 약하므로 보관에 주의하십시오.

**사건 번호**: 0004PSYA.2025.가04

**담당**: 류재관 (HMU-003)

---

**1. 상황 요약**

고택에서 봉인 부적 손상이 발견되어 긴급 출동하였습니다.

**2. 조사 결과**

기존 부적은 약 15년 경과로 효력이 약화된 상태였습니다.
종이 재질이 손상되고 주사가 탈색되어 재시공이 필요하다고 판단했습니다.

**3. 조치 사항**

신규 부적을 황지에 주사로 제작하여 설치했습니다.
적절한 경문을 선택하였으며, 결계와 연동하여 안정화했습니다.
